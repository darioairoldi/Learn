# Prompt engineering artifact dependency map

Purpose: dependency-tracking entry point for PE artifacts.

Referenced by: `pe-meta-update`, `meta-validator`, `pe-context-builder`, and dependency-aware review workflows.

## Quick use workflow

1. Start here to classify expected blast radius.
2. Open full appendix data for detailed inbound and outbound mappings.
3. Apply impact class before proposing changes.
4. Revalidate dependents based on class.

## Impact classification

| Dependents | Impact | Required action |
|---|---|---|
| 0–2 | Low | Apply and validate changed artifact plus direct dependents |
| 3–5 | Medium | Review dependent set before apply; validate incrementally |
| 6+ | High | Create explicit rollout plan and staged validation gates |
