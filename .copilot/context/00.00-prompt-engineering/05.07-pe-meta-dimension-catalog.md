## Dimension-to-handler mapping

| Type | Handler | Why |
|---|---|---|
| Deterministic (D1-D4) | pe-meta-validator with deterministic tools | Parse YAML, count lines, resolve paths |
| Structural (D5, D14) | pe-meta-validator with standard model | Pattern matching against known structures |
| Semantic (D6-D11, D16-D19) | pe-meta-validator with reasoning model | Requires understanding meaning |
| External (D12-D13) | pe-meta-researcher with `fetch_webpage` | Requires accessing external sources |
