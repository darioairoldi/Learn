# pe-meta Per-Type Validation Checklists

## How to use

The pe-meta-validator loads this file and dispatches to the appropriate section based on the target artifact's file path:
- `.copilot/context/` → Context file checklist
- `.agent.md` → Agent checklist
- `.prompt.md` → Prompt checklist
- `.instructions.md` → Instruction file checklist
- `SKILL.md` → Skill checklist
- `.template.md` → Template checklist
- `.json` in `hooks/` → Hook checklist
- `prompt-snippets/` → Snippet checklist
