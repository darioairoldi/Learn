**Naming convention checklist (all three new names):**

- ✅ Numeric prefix `NN.NN-` matches directory convention
- ✅ Kebab-case body, lowercase ASCII only
- ✅ `.md` extension
- ✅ Descriptive of file role (index / catalog / guide)
- ✅ Natural sort order preserved (index sorts first; catalog and maintenance follow governance peers)
- ✅ No collision with existing files (00.01/00.02/00.03 occupied — 00.00/00.04/00.05 are free)

## 🔙 Rollback plan

The change is **fully reversible via `git revert`** because every action is contained in a single commit.

- **Soft rollback (before commit):** `git restore .` (discards all in-flight edits) + `git mv` reverses for the three files (`git mv 00.00-context-structure-index.md STRUCTURE-README.md`, etc.).
- **Hard rollback (after commit):** `git revert <commit-sha>` produces a single revert commit that undoes the rename and all 228 back-reference updates atomically.
- **Partial rollback is not supported** — the rename + back-reference updates are a unit. If a single back-reference update has a problem, fix it in a follow-up commit on the same branch; do not partially revert.
