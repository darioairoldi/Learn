```markdown
## MCP Apps â€” Rich UI in chat (experimental)

MCP servers can go beyond text responses by returning interactive HTML UIs directly in the chat panel. The `@anthropic-ai/ext-apps` package enables this through a `ui://` resource scheme.

### How it works

1. The MCP server declares a `ui://` resource using the ext-apps framework
2. The tool returns a reference to the UI resource instead of plain text
3. The chat client renders the HTML/CSS/JS inline in the conversation
4. A promise-based pattern allows the server to wait for user interaction before continuing

### Key implementation details

- **Package:** `@anthropic-ai/ext-apps` (TypeScript)
- **Resource scheme:** `ui://your-app-name`
- **Bundling:** Use Vite to bundle the UI into a single HTML file
- **Promise pattern:** Server suspends execution while waiting for user interaction
- **Status:** Experimental â€” API may change

### When to use MCP Apps

- Interactive forms or configuration wizards inline in chat
- Data visualizations that go beyond markdown tables
- Multi-step workflows requiring user choices before proceeding

> For a practical walkthrough, see [Burke Holland â€” MCP Apps](../../../01.00-news/20260214.4-burke-holland-mcp-apps/summary.md).
```
